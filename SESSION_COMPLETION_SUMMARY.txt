PROBLEM IDENTIFIED
================================================================================

Error Location: src/app/api/user/update/route.ts:29
Error Message: TypeError: Cannot read properties of undefined (reading 'findOneAndUpdate')

Root Cause:
- User model exports as: export const User = ...
- But code imported as: const { UserModel } = await import("@/models/user")
- Result: UserModel = undefined
- When used: undefined.findOneAndUpdate() → TypeError

================================================================================
